研修用ブログ｜REST API

ブログ記事関連

API名 HTTPメソッド パス 必要なパラメータ ステータスコード 説明
GET articles GET /api/articles/ user-token 200（OK） 認証済ユーザーのブログ記事一覧を取得
POST articles POST /api/articles/ user-token, article-title, article-body 201（Created） 認証済ユーザーで新規ブログ記事を作成
GET article GET /api/articles/ user-token, article-id 200（OK） 認証済ユーザーのブログ記事を1件取得
PUT articles PUT /api/articles/ user-token, article-id, article-title, article-body 200（OK） 認証済ユーザーのブログ記事を更新
DELETE articles DELETE /api/articles/ user-token, article-id 200（OK） 認証済ユーザーのブログ記事を削除

ユーザー情報関連

API名 HTTPメソッド パス 必要なパラメータ ステータスコード 説明
GET users GET /api/users/ user-name, user-password 200（OK） ユーザー認証＆トークン取得
POST users POST /api/users/ user-name, user-password 201（Created） 新規ユーザー登録
GET user GET /api/users/ user-token 200（OK） ユーザー情報（ID, ユーザー名）を取得
